## How to run
To Run this, install termux in your mobile.

type:
```bash
apt update -y && apt upgrade -y
apt install python git -y
git clone https://github.com/opsonusdh/Google-homepage-for-mobile
cd Google-homepage-for-mobile
pip install flask
python app.py
```
